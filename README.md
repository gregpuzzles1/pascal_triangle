# Pascal’s Triangle Visualizer

A sleek, interactive visualization of Pascal’s Triangle with responsive row limits, hockey-stick highlighting, and customizable display controls.

## Quickstart

- Install dependencies: `npm install`
- Start dev server: `npm run dev`
- Build for production: `npm run build`
- Preview build: `npm run preview`

## Deployment

This project deploys to GitHub Pages via GitHub Actions from the `main` branch only.

## Configuration

Footer links are configured in `src/lib/links.ts`. Update the repository and issue URLs there if needed.
